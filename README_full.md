## ΓÜí Quick Start

Get started in under 2 minutes:

```bash
# 1∩╕ÅΓâú Clone repository
git clone https://github.com/mohamednoorulnaseem/airfoil-rl-optimizer.git
cd airfoil-rl-optimizer

# 2∩╕ÅΓâú Create virtual environment (recommended)
python -m venv .venv
source .venv/bin/activate  # On Windows: .venv\Scripts\activate

# 3∩╕ÅΓâú Install package
pip install -e .

# 4∩╕ÅΓâú Train your first model (50K timesteps, ~10 minutes)
python train_rl.py --total-timesteps 50000

# 5∩╕ÅΓâú Launch interactive web interface
python app.py
# ≡ƒîÉ Open http://127.0.0.1:8050 in your browser
```

### ≡ƒÄ« One-Liner Demo

```python
from src.optimization.rl_agent import AirfoilRLAgent
from src.optimization.multi_objective_env import MultiObjectiveAirfoilEnv

# Load pre-trained agent
env = MultiObjectiveAirfoilEnv()
agent = AirfoilRLAgent(env, model_path="models/ppo_airfoil_final.zip")

# Optimize airfoil
obs, _ = env.reset()
action, _ = agent.predict(obs)
print(f"Optimized parameters: {action}")  # [camber, position, thickness]
```

- Γ£à **CFD Integration** - XFOIL panel method + Stanford SU2 RANS solver
- Γ£à **Physics-Informed ML** - PINN surrogate achieving 62% speedup with <2% accuracy loss

## ≡ƒö¼ Technical Approach

### ≡ƒñû Reinforcement Learning Framework

<div align="center">

```mermaid
sequenceDiagram
    participant Agent as PPO Agent
    participant Env as Environment
    participant CFD as CFD Solver
    participant Mfg as Manufacturing

    Agent->>Env: Action [m, p, t]
    Env->>CFD: Evaluate aerodynamics
    CFD-->>Env: Cl, Cd, L/D
    Env->>Mfg: Check constraints
    Mfg-->>Env: Feasibility score
    Env->>Agent: Reward + Next State
    Agent->>Agent: Update policy
```

</div>

#### PPO Configuration

- **Algorithm:** Proximal Policy Optimization (Schulman et al., 2017)
- **Framework:** Stable-Baselines3 2.7+ with Gymnasium
- **Training:** 100,000 timesteps (~10 hours on single GPU)
- **Batch size:** 2048 samples
- **Learning rate:** 3├ù10Γü╗Γü┤ with annealing
- **Discount factor (╬│):** 0.99
- **GAE lambda (╬╗):** 0.95

#### Multi-Objective Reward Function

```python
R = 0.40 ├ù (L/D - L/D_baseline) / L/D_baseline      # Aerodynamic efficiency
  + 0.25 ├ù (Cl_max - Cl_baseline) / Cl_baseline    # High-lift performance
  + 0.20 ├ù Stability_margin                         # Pitching moment control
  + 0.15 ├ù Manufacturing_feasibility                # Production constraints
  - 0.10 ├ù Penalty_violations                       # Hard constraint penalties
```

### ≡ƒîè Physics-Informed Neural Networks

Our PINN surrogate model combines:

- **Data-driven learning:** Trained on 500+ XFOIL evaluations
- **Physics constraints:** Navier-Stokes residuals in loss function
- **Uncertainty quantification:** Bayesian neural network with dropout

**Architecture:**

```
Input (3 params) ΓåÆ Dense(128) ΓåÆ Dense(256) ΓåÆ Dense(256) ΓåÆ Dense(128) ΓåÆ Output (Cl, Cd)
                    Γåô Physics loss
              Navier-Stokes PDE residuals
```

**Performance:**

- Γ£à **62% speedup** vs XFOIL (0.01s vs 0.026s per evaluation)
- Γ£à **<2% mean error** on Cl predictions
- Γ£à **<5% mean error** on Cd predictions
- Γ£à **Uncertainty bounds** provided for all predictions
